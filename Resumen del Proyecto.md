# Future Prospects Artists

### Plataforma web para la promoción y monetización de artistas visuales no conocidos.

## Descripción del Proyecto
"Future Prospects Artists" es una solución tecnológica diseñada para cerrar la brecha entre el talento artístico de alto nivel y la visibilidad comercial. A diferencia de las redes sociales masivas, esta plataforma se enfoca en la curatela y exhibición de portafolios profesionales, permitiendo que artistas no conocidos logren sus primeras ganancias económicas a través de un sistema de contacto directo para comisiones.

## El Problema
Muchos artistas con habilidades técnicas excepcionales permanecen en el anonimato debido a algoritmos que favorecen la cantidad de contenido sobre la calidad técnica. Según los beneficiarios: *"No existe un espacio que valide el nivel profesional y conecte directamente con clientes reales"*.

## Tecnologías Utilizadas
Este proyecto utiliza el **Stack MERN** para garantizar escalabilidad y rendimiento:
- **Frontend:** React.js (Interfaz dinámica y responsiva)
- **Backend:** Node.js & Express (API RESTful)
- **Base de Datos:** MongoDB (Gestión flexible de metadatos)
- **Almacenamiento:** Cloudinary (Hosting de imágenes en alta resolución)
- **Gestión:** Jira (Metodología Scrum)

## Características Principales
- **Galería High-Res:** Visualización de arte sin pérdida de calidad.
- **Tiempos de Carga Optimizados:** Respuesta menor a 3 segundos.
- **Sistema de Comisiones:** Conexión directa entre artista y cliente.
- **Portafolios Personalizados:** Espacio profesional para cada creador.

## Metodología de Desarrollo
Se implementa la metodología ágil **Scrum**, con Sprints semanales para asegurar la entrega incremental de valor y trazabilidad total entre los objetivos planteados y el código desarrollado.
